Maneiras de utilizar e/ou adicionar css

Temos maneiras de utilizar o css ao html.

  1. Pelo inline: quando os estilos são adicionados por um atributo;
  2. Pelo internal: quando o CSS é adicionado na tag head;
  3. Pelo external: quado o css é adicionado através de um arquivo externo, e depois importado no HTMl;

    4. Qual vamos escolher?

    Vamos optar sempre pelo external CSS, quando houver a opção.

    A anatomia do CSS

    Com CSS aplicando CSS a um elemento. Primeiramente devemos selecionar o elemento, isso pode ser feito através da tag do elemento. Depois precisamos colocar as propriedades e os valores, se quisermos mudar a cor de algo utilizamos: color: red, nome da propriedade, dois pontos, valor, ponto e vírgula.

    Inline CSS

    O inline CSS é quando você aplica estilos diretamente em um elemento HTML usando o atributo style, mas pode ser utilizado sem selecionar o elemento, pois é uma tributo diretamente inserido no mesmo.

    Ele nos permite:

  4. Aplicar estilos específicos a um único elemento;
  5. Sobrepor estilos definidos em folhas de estilo externas ou internas;
  6. Ser útil para testes rápidos ou ajustes pontuais;

    7. Porém, o inline CSS não é recomendado para uso extensivo, pois pode tornar o código HTML desorganizado e difícil de manter. É melhor usar folhas de estilo externas ou internas para aplicar estilos de forma consistente em todo o site.

    Internal CSS

    O CSS interno é uma técnica melhor que o inline, vamos colocar todos os estilos na tag head;

  7. As regras precisam também estar entre a tag style;
  8. E através desta maneira, é necessário selecionar o elmento alvo: p{ color: red;}

    External CSS

    Para adicionar CSS com esta técnica precisamos criar um arquivo . css;

  9. Geralmente eles ficam numa pasta chamada css;
  10. E nós o importamos através da tag link;
  11. O link deve estar dentro da tag head;
  12. O link deve ter o atributo rel com o valor stylesheet e as regras que estão no arquivo são aplicadas no HTML.

    13. Múltiplos arquivos CSS

    É possível ter múltiplos arquivos CSS em um projeto. Para isso, basta criar vários arquivos .css e importá-los na tag head do HTML.

    Por exemplo, você pode ter um arquivo style.css para estilos gerais, outro chamado titles.css para estilos de títulos, e assim por diante.

    Isso ajuda a organizar melhor o código e facilita a manutenção, especialmente em projetos

    Comentários no CSS

  13. Comentários do CSS são como os de HTML, utilizamos para descrever algo no código;
  14. Caso o código precise de explicações adicionais, podemos usar comentários para esclarecer;
  15. A sintaxe é: /* algum comentário */, isso caso o código seja inspencionado, aí os comentários serão exibidos

    16. Classes e ids

  16. Classes e ids são seletores que nos permitem aplicar estilos a elementos específicos;
  17. Um id deve ser único dentro de uma página, enquanto várias classes podem ser aplicadas a um ou mais elementos;
  18. A sintaxe para usar um id é: #meuId { color: red; }, e para uma classe: .minhaClasse { color: blue; };

    19. Classes e ids são fundamentais para aplicar estilos específicos a elementos HTML. Eles nos permitem personalizar a aparência de partes específicas do site, tornando o design mais flexível e organizado.

    Classes

    As classes são usadas para agrupar elementos que compartilham estilos semelhantes. Você pode aplicar a mesma classe a vários elementos, facilitando a manutenção e a consistência do design.

    Para usar uma classe, você a define no CSS com um ponto (.) antes do nome da classe. Por exemplo:

    .minhaClasse {
                color: blue;
            }

    Em seguida, você aplica a classe aos elementos HTML usando o atributo class:

    <div class="minhaClasse">Este é um exemplo de div com classe.</div>

    Você pode aplicar várias classes a um único elemento, separando os nomes por espaços:

    <div class="classe1 classe2">Este é um exemplo de div com múltiplas classes.</div>
