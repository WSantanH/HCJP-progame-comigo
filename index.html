Maneiras de utilizar e/ou adicionar css

Temos maneiras de utilizar o css ao html.

  • Pelo inline: quando os estilos são adicionados por um atributo;
  • Pelo internal: quando o CSS é adicionado na tag head;
  • Pelo external: quado o css é adicionado através de um arquivo externo, e depois importado no HTMl;

Qual vamos escolher?

Vamos optar sempre pelo external CSS, quando houver a opção.

A anatomia do CSS

Com CSS aplicando CSS a um elemento. Primeiramente devemos selecionar o elemento, isso pode ser feito através da tag do elemento. Depois precisamos colocar as propriedades e os valores, se quisermos mudar a cor de algo utilizamos: color: red, nome da propriedade, dois pontos, valor, ponto e vírgula.

Inline CSS

O inline CSS é quando você aplica estilos diretamente em um elemento HTML usando o atributo style, mas pode ser utilizado sem selecionar o elemento, pois é um atributo diretamente inserido no mesmo.

Ele nos permite:

  • Aplicar estilos específicos a um único elemento;
  • Sobrepor estilos definidos em folhas de estilo externas ou internas;
  • Ser útil para testes rápidos ou ajustes pontuais;

Porém, o inline CSS não é recomendado para uso extensivo, pois pode tornar o código HTML desorganizado e difícil de manter. É melhor usar folhas de estilo externas ou internas para aplicar estilos de forma consistente em todo o site.

Internal CSS

O CSS interno é uma técnica melhor que o inline, vamos colocar todos os estilos na tag head;

  • As regras precisam também estar entre a tag style;
  • E através desta maneira, é necessário selecionar o elemento alvo: p{ color: red; }

External CSS

Para adicionar CSS com esta técnica precisamos criar um arquivo .css;

  • Geralmente eles ficam numa pasta chamada css;
  • E nós o importamos através da tag link;
  • O link deve estar dentro da tag head;
  • O link deve ter o atributo rel com o valor stylesheet e as regras que estão no arquivo são aplicadas no HTML.

Múltiplos arquivos CSS

É possível ter múltiplos arquivos CSS em um projeto. Para isso, basta criar vários arquivos .css e importá-los na tag head do HTML.

Por exemplo, você pode ter um arquivo style.css para estilos gerais, outro chamado titles.css para estilos de títulos, e assim por diante.

Isso ajuda a organizar melhor o código e facilita a manutenção, especialmente em projetos grandes.

Comentários no CSS

  • Comentários do CSS são como os de HTML, utilizamos para descrever algo no código;
  • Caso o código precise de explicações adicionais, podemos usar comentários para esclarecer;
  • A sintaxe é: /* algum comentário */, isso caso o código seja inspecionado, aí os comentários serão exibidos

Classes e ids

  • Classes e ids são seletores que nos permitem aplicar estilos a elementos específicos;
  • Um id deve ser único dentro de uma página, enquanto várias classes podem ser aplicadas a um ou mais elementos;
  • A sintaxe para usar um id é: #meuId { color: red; }, e para uma classe: .minhaClasse { color: blue; };

Classes e ids são fundamentais para aplicar estilos específicos a elementos HTML. Eles nos permitem personalizar a aparência de partes específicas do site, tornando o design mais flexível e organizado.

Classes

As classes são usadas para agrupar elementos que compartilham estilos semelhantes. Você pode aplicar a mesma classe a vários elementos, facilitando a manutenção e a consistência do design.

Para usar uma classe, você a define no CSS com um ponto (.) antes do nome da classe. Por exemplo:

.minhaClasse {
    color: blue;
}

Em seguida, você aplica a classe aos elementos HTML usando o atributo class:

<div class="minhaClasse">Este é um exemplo de div com classe.</div>

Você pode aplicar várias classes a um único elemento, separando os nomes por espaços:

<div class="classe1 classe2">Este é um exemplo de div com múltiplas classes.</div>

Ids

Os ids são usados para identificar elementos únicos na página. Para usar um id, defina no CSS com # antes do nome:

#meuId {
    color: red;
}

E aplique ao elemento HTML:

<div id="meuId">Este é um exemplo de div com id.</div>
