Java

  • Orientação a Objetos
  • Desenvolvimento Desktop e Web
  • Frameworks populares: Spring, Hibernate
  • Exemplo de código:
public class OlaMundo {
    public static void main(String[] args) {
        System.out.println("Olá, mundo Java!");
    }
}
Dicas para aprender Java:

• Entenda bem os conceitos de orientação a objetos: classes, objetos, herança, polimorfismo e encapsulamento.
• Pratique bastante com exercícios de lógica e algoritmos.
• Use IDEs como IntelliJ ou Eclipse para facilitar o desenvolvimento.
• Aprenda sobre tratamento de exceções para evitar erros inesperados.
• Explore as bibliotecas padrão do Java, como Collections, Streams e IO.
• Para web, estude Servlets, JSP e frameworks como Spring Boot.
• Leia e escreva códigos comentados para facilitar a manutenção.
• Participe de fóruns e comunidades como Stack Overflow e grupos no Telegram.
• Faça projetos práticos, como sistemas de cadastro, calculadoras, ou APIs REST.
• Sempre revise e refatore seu código para deixá-lo mais limpo e eficiente.

Python

  • Sintaxe simples e poderosa
  • Automação, Web, Data Science
  • Frameworks populares: Django, Flask
  • Exemplo de código:
def ola_mundo():
    print("Olá, mundo Python!")

ola_mundo()
Mais exemplos de código Python:

Variáveis e tipos: 
x = 10
nome = "Maria"
pi = 3.14
Estruturas de decisão: 
if x > 5:
    print("x é maior que 5")
else:
    print("x não é maior que 5")
Laços de repetição: 
for i in range(5):
    print(i)
Funções: 
def soma(a, b):
    return a + b

print(soma(2, 3))
Listas: 
lista = [1, 2, 3, 4]
for item in lista:
    print(item)
Dicionários: 
dados = {"nome": "Ana", "idade": 22}
print(dados["nome"])
Automação de arquivos: 
with open("arquivo.txt", "w") as f:
    f.write("Olá, mundo!")
Web com Flask: 
from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
    return "Olá, Flask!"

JavaScript

  • Manipulação de DOM
  • Eventos e funções
  • Frameworks populares: React, Vue, Node.js
  • Exemplo de código:
function olaMundo() {
    alert("Olá, mundo JavaScript!");
}
olaMundo();
Mais exemplos de código JavaScript:

Variáveis: 
let x = 10;
const nome = "Maria";
var pi = 3.14;
Estruturas de decisão: 
if (x > 5) {
    console.log("x é maior que 5");
} else {
    console.log("x não é maior que 5");
}
Laços de repetição: 
for (let i = 0; i < 5; i++) {
    console.log(i);
}
Funções: 
function soma(a, b) {
    return a + b;
}
console.log(soma(2, 3));
Arrays: 
let lista = [1, 2, 3, 4];
lista.forEach(function(item) {
    console.log(item);
});
Objetos: 
let dados = {nome: "Ana", idade: 22};
console.log(dados.nome);
Manipulação de DOM: 
document.getElementById("meuElemento").innerText = "Novo texto";
Eventos: 
document.getElementById("botao").onclick = function() {
    alert("Botão clicado!");
};
Função Arrow: 
const soma = (a, b) => a + b;
console.log(soma(2, 3));
